About Soundlab

Soundlab is a software audio player that will transform the way you experience digital music. With a sleek and intuitive interface, Soundlab is designed to be easy to use while offering advanced features for music lovers.

Easy to install

Check our Github page to learn how to install the application

Made with Python

Soundlab was my by Python and the GUI toolkit that we are using is Tkinter

Soundlab Features

Soundlab supports a wide range of audio file formats, including MP3, FLAC, AAC, WAV, and more. Create playlists, browse your music library with ease, and take advantage with features like shuffle, playback, and replay gain.

  • Add multiple MP3 files
  • Play, pause music
  • Forward, backward song
  • Friendly interface
