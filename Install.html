How To Install Soundlab

Please visit: Github link to download Soundlab application.

After that, click on the code button and download the zip file:

Extract the zip file and run it on IDE(We suggest using Pycharm). Go to the command line and install these packages:
  • Pygame
  • Pillow
  • eyed3
  • mutagen
  • tk
  • coverage
  • packaging
The command to install package: "pip install (package-name)", e.g: "pip install Pillow"
After that, run the main.py file and enjoy Soundlab
If you have any issues, try to leave a comment on the Github project
